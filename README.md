# Zoo Upgrade System 🐾

A Java-based console application that simulates zoo management using Object-Oriented Programming concepts.

## Features
- Add animals
- Different behaviors (Eat, Swim, Walk)
- File handling
- Animal-specific data
- Interactive menu-driven program

## Tech Used
- Java
- OOP (Inheritance, Interfaces, Abstraction)
- File I/O
- Serialization-Deserialization
- Erros and exception handling

## How to Run
javac Main.java
java Main
